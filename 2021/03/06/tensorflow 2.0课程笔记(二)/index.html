逻辑回归与交叉熵

线性回归是连续值，逻辑回归给的是“是”与“否”，可以看做是0和1，是一个分类问题。

逻辑回归用的函数为sigmoid函数。

sigmoid函数：将结果映射到0~1之间，看做是一个概率分布

使用sigmoid，mse不再适用于较小值的损失比较，需要很大的迭代次数，因此分类问题用交叉熵损失函数。

交叉熵损失函数：刻画的是概率间距，交叉熵越小，概率分布越接近。可以放大概率间的损失

二元交叉熵（binary_crossentropy）

逻辑回归实现

import tensorflow as tf

import pandas as pd

import numpy as np

import matplotlib as plt

%matplotlib inline

  • 导入库

data = pd.read_csv('dataset/credit-a.csv',header=None)

data.head()

  • 无表头数据，第一行就是数据，最后一列为目标值（-1和1）

data.iloc[:,-1].value_counts()

  • 统计二分类个数

x = data.iloc[:,:-1]

y = data.iloc[:,-1].replace(-1,0)

  • 把目标值的-1替换成0

model =tf.keras.Sequential()

model.add(tf.keras.layers.Dense(4,input_shape(15,),activation='relu'))

model.add(tf.keras.layers.Dense(4,activation='relu'))

model.add(tf.keras.layers.Dense(1,activation='sigmoid'))

  • 两个隐藏层（4个隐藏单元数，自己设计的），一个输出层

model.summary()

model.compile(optimizer='adam',

loss='binary_crossentropy',

metrics=['acc']

)

  • 损失函数为binary_crossentropy交叉熵，过程中输出正确率acc

history = model.fit(x,y,epochs=100)

  • 训练过程

history.history.keys()

  • 输出为dict_keys([“loss”,”acc”])

plt.plot(history.epoch,history.history.get('loss'))

  • 绘制图形loss随训练变化的趋势

softmax多分类

softmax：是对数几率回归在N个可能不同的值上的推广，要求每个样本必须属于某个类别，且所有可能的样本均被覆盖，样本分量之和为1

tf.keras里面用categorical_crossentropy和sparse_categorical_crossentropy来计算softmax交叉熵

Fashion MNIST数据集：相对较小，测试和调试代码的良好起点。6W张图像训练，1W张图像评估

多分类实现

import tensorflow as tf

import pandas as pd

import numpy as np

import matplotlib as plt

%matplotlib inline

  • 导入库

(train_image,train_label),(test_image,test_label) = tf.keras.datasets.fashion_mnist.load_data()

  • 下载数据集

plt.imshow(train_image[0])

train_image.shape

  • 查看图片，train_image为(60000,28,28)

train_image = train_image/255

test_image = test_image/255

  • 数据归一化

model = tf.keras.Sequential()

model.add(tf.keras.layers.Flatten(input_shape=(28,28))) #28*28

model.add(tf.keras.layers.Dense(128,activation='relu'))

model.add(tf.keras.layers.Dense(10,activation='softmax'))

  • Flatten层将二维数据展开成一维
  • 隐藏层参数128，要求层数足够大
  • 输出层参数10表示输出10个概率，分为了10类。和二分类不同，二分类参数为1表示直接输出0或1

model.compile(optimizer='adam',

loss='sparse_categorical_crossentropy'

metrics=['acc']

)

  • 列表（label）使用了顺序编码的时候使用sparse_categorical_crossentropy

model.fit(train_image,train_label,epochs=5)

model.evaluate(test_images,test_label)

独热编码与交叉熵损失函数

独热编码one-hot：[1,0,0]、[0,1,0]、[0,0,1]

train_label_onehot = tf.keras.utils.to_categorical(train_label)

test_label_onehot = tf.keras.utils.to_categorical(test_label)

  • 转换成one-hot编码

model = tf.keras.Sequential()

model.add(tf.keras.layers.Flatten(input_shape=(28,28))) #28*28

model.add(tf.keras.layers.Dense(128,activation='relu'))

model.compile(optimizer='adam',

loss='categorical_crossentropy'

metrics=['acc']

)

  • 模型建立及编译，loss修改为categorical_crossentropy

model.fit(train_image,train_label_onehot,epochs=5)

model.evaluate(test_images,test_label_onehot)
