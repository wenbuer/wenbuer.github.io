HTB VPN配置

服务器： 阿里云 ubuntu

在服务器安装openvpn

apt-get update

apt-get install openvpn

在HTB的Labs页面中找到连接Starting Point

选择好其中一个节点进行下载.ovpn文件

在服务器用openvpn连接到staring point

openvpn starting_point_XXXXX.ovpn

点击Archetype机器，选择Jion Machine，加入之后会告诉你该机器的IP地址

在服务器ping该地址，可以ping通

测试VPN已经搭建完成
